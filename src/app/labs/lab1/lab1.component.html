In this page, the considered and implemented web usability of the portfolio website is detailed.

The navbar items gives the access to users to easily navigate to the page they need.

When user move the curse to the text, the color of button text chaning to show that it is button and it is clickable.

The portfolio website has a minimal design for presenting 5 labs and 3 assignmnets.

5. Consistency:

I tried to keep the design in each page consistet and related to each other.

6. Guidance:

Besides nav items, for example Labs in navbar, there is small but visible arrow to shoe that the Labs is dropdown and user can click to see more options.

The portfolio website is very clean and easy to navigate with familiar layout which helps people to quickly find what they are looking for.

8. Credibility:

There is a Copyright section in footer and also in Home page user can see the student oath which creates a trust.

9. Relevancy:

The website is clear but also contents are also relevant and users know why they visit the site.

There is a nice balance between content and graphics on website. There' is a graphic design on home page to attract user's attention.
