The practice with video editing and conversion with different formats is demonstrated on this page.

Video: My Trip to Athens, Greece

" Eleftheria i thanatos (Greek: Ελευθερία ή θάνατος, IPA: [elefθeˈri.a i ˈθanatos]; 'Freedom or Death') is the motto of Greece. "

Reflection:

Explain the work process, including editing on music and video.
1. I decided to use Canva Website as my editing tool since it has a good UI & UX and I was working with Canva for a long time.
2. To create a video I used short videos of my trip to Athens. I just put them together, changed the lenght of each video and removed the noise.
3. For each video I added a text to describe it. To make it more intersting I added a transition to the text.
4. I used a local greek music that is easy to hear and related to the video.

Explain your thoughts on the video creation, such as file size and file format.
File Size: The reason for making the file size as small as possible is essential to make sure that it is readily available to users. Larger video files can create an extra load on the website and make things slower for users when trying to access the content.
File Format:The format I chose is MP4, so it is more compatible with a wide range of platforms and players. It is the default for web and social media because it produces smaller file sizes with outstanding quality.
